Сократите рутинную работу с объявлениями
  • Перед размещением объявлений Сделка проверит их на соответствие требованиям каждой рекламной площадки, а также правильность заполнения полей
  • После размещения объявлений — сообщит об изменениях статуса объектов: фид ещё не запрашивался / фид запрошен/ ожидает размещения/ размещено / ошибка
  • При выявлении ошибки — покажет именно те поля, где требуется внести изменения
  • Отслеживайте объявления с низкими результатами просмотров в списке «Реклама требующая внимания»
  • Просматривайте все объекты в отдельном списке с удобными фильтрами

Отслеживайте результаты запусков рекламы на отдельной странице

  • Статистика со всех рекламных площадок

  • Стоимость размещения каждого объявления
  • Удобные фильтры для поиска объектов: статус размещения / доска объявлений / ответственный / кто запустил в рекламу / объекты с агентским договором и т.д.
  • Уведомление руководителю о статусе размещения объявлений: ожидает размещения / размещено / реклама завершена / требует визирования

Коммуникации с клиентами и подрядчиками в одном окне

  • Принимайте сообщения из WhatsApp, Telegram, Instagram, VK, заявки с вашего сайта и Авито в одном кабинете Сделки (окне CRM)
  • Заявки и объекты клиента, история переписки, записи разговоров внутри одной карточки
  • Ищите контакты с помощью удобных фильтров по типу клиента или сделки / этапу сделки / объекту / дате создания карточки
  • Присваивайте статус каждому контакту: клиент / агент другого АН / инвестор / рабочий или личный контакт / нотариус / юрист / фотограф / клининг
  • Система не допустит дублирование контактов
  • Клиентская база защищена от несогласованного скачивания

Создавайте свою воронку продаж, статусы и поля, условия, скрипты и рекомендации

  • При работе с заявкой система будет котнролировать последовательность и правильность работы сотрудника, подключать новых сотрдуников и выставлять задачи
  • Сделка фиксирует каждое действие сотрудников, проверяет результат, даёт подсказки и задания
  • Каждое действие фиксируется, имеет свой результат и ведёт к заявке или завершению сделки.
  • Риэлтор не сможет безрезультатно совершать действия, потенциально приводящие к заявке.
